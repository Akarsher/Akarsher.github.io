# Personal Portfolio

A responsive personal portfolio website showcasing my projects, skills, and experience.

## Features

- **Responsive Design**: Works on all devices and screen sizes
- **Dark/Light Theme**: Toggle between day and night modes
- **Interactive Project Cards**: Flip cards with project details and repository links
- **Smooth Animations**: AOS (Animate On Scroll) library integration
- **Contact Form**: Email integration for direct contact
- **Modern UI/UX**: Clean and professional design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll)
- EmailJS
- Boxicons

## Sections

1. **Home**: Introduction and social links
2. **About**: Personal information and background
3. **Skills**: Technical skills organized by categories
4. **Projects**: Portfolio of completed projects with repository links
5. **Contact**: Contact form and information
